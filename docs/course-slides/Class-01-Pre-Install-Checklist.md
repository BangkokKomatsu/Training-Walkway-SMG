# เตรียมตัวก่อนคลาส 1 (Pre-Class Checklist)

> ส่งเอกสารนี้ให้ผู้เรียนอ่าน**ก่อนวันเรียน** — เพื่อเช็คความพร้อมเครื่องล่วงหน้า จะได้ไม่เสียเวลาในคลาสไปกับปัญหาที่แก้ได้ตั้งแต่ก่อนเข้าเรียน
> ⚠️ **ไม่ต้องติดตั้งโปรแกรมใด ๆ ล่วงหน้า** — วันเรียนจะติดตั้งพร้อมกันทั้งหมดแบบ live ในคลาส เอกสารนี้แค่เช็คว่า "เครื่องพร้อมให้ติดตั้งได้" เท่านั้น

## 1. เครื่อง/OS

- [ ] Windows 10 หรือ 11 (หรือ macOS/Ubuntu ก็ได้ แต่คำสั่งในคลาสจะสาธิตบน Windows เป็นหลัก)
- [ ] พื้นที่ disk ว่าง **อย่างน้อย 5 GB** (โมเดล AI และ library ที่จะติดตั้งมีขนาดใหญ่ โดยเฉพาะ PyTorch)
- [ ] RAM แนะนำ 8 GB ขึ้นไป (ใช้ได้กับเครื่องทั่วไป ไม่จำเป็นต้องมีการ์ดจอ/GPU)

## 2. สิทธิ์ติดตั้งโปรแกรม (สำคัญที่สุด)

- [ ] เครื่องที่จะใช้เรียนต้อง**ติดตั้งโปรแกรมเองได้** (มีสิทธิ์ Administrator หรือได้รับอนุญาตจาก IT ให้ลงโปรแกรมได้)
- **วิธีเช็คเบื้องต้น (Windows):** ลองเปิด Control Panel → Programs หรือลองดาวน์โหลดโปรแกรมอะไรสักตัวมาติดตั้งดู ถ้าเจอ popup ให้ใส่รหัสผ่าน Admin ที่ไม่มี แปลว่าเครื่องถูกล็อกไว้
- **ถ้าใช้โน้ตบุ๊กบริษัทที่ล็อกสิทธิ์ไว้:** ต้องติดต่อฝ่าย IT ของบริษัทเพื่อขอสิทธิ์ติดตั้งโปรแกรม (Python, VS Code) **ก่อนวันเรียน** — แจ้งผู้ดูแลคอร์สล่วงหน้าอย่างน้อย 2-3 วันถ้าคาดว่าจะติดปัญหานี้ เพราะการขอสิทธิ์ IT บริษัทมักใช้เวลา
- ถ้าไม่มีสิทธิ์และขอไม่ทันวันเรียน: แจ้งผู้สอนก่อนคลาสเริ่ม จะได้เตรียมแผนสำรอง (เช่น ให้ดูจากเพื่อนก่อน แล้วนัดติดตั้งรอบเสริม)

## 3. อินเทอร์เน็ต

- [ ] อินเทอร์เน็ตเสถียร ความเร็วพอสมควร — ระหว่างคลาสจะดาวน์โหลด library รวมกันหลายร้อย MB (โดยเฉพาะ PyTorch ที่ใหญ่ที่สุด)
- [ ] ถ้าอยู่หลัง VPN/Proxy ของบริษัท: เช็คว่า `pip install` และการดาวน์โหลดจากอินเทอร์เน็ตทั่วไปใช้งานได้ปกติ (บางบริษัท proxy บล็อก pypi.org) — ถ้าไม่แน่ใจให้ทดลองเปิดเว็บ <https://pypi.org> ดูก่อนว่าเข้าได้ไหม

## 4. บัญชี/สิ่งที่ต้องเตรียม

- [ ] **ไม่ต้องมีบัญชี GitHub** (คลาสนี้ไม่สอน/ไม่ใช้ Git)
- [ ] ได้รับไฟล์โปรเจกต์ (โฟลเดอร์ repo) มาไว้ในเครื่องแล้ว หรือรอรับใน chat วันเรียน
- [ ] Teams/ช่องทางประชุมออนไลน์ที่ใช้เรียน — ทดสอบเข้าได้ก่อนวันจริง (กล้อง/ไมค์ใช้งานได้)

## 5. พื้นฐานความรู้ (ไม่ต้องกังวล)

- ✅ **ไม่จำเป็นต้องเขียนโค้ด/เขียน Python มาก่อน** — คลาสจะปูพื้นฐานให้ในช่วงท้าย
- ✅ **ไม่จำเป็นต้องมีความรู้ AI/Machine Learning มาก่อน**
- ✅ ใช้คอมพิวเตอร์พื้นฐานเป็น (เปิดโปรแกรม, copy/paste, เปิด terminal) ก็เพียงพอ

## 6. สรุป Checklist ก่อนเข้าคลาส

- [ ] เครื่องมีสิทธิ์ติดตั้งโปรแกรมได้ (หรือขอสิทธิ์ IT เรียบร้อยแล้ว)
- [ ] Disk ว่าง ≥ 5 GB
- [ ] อินเทอร์เน็ตพร้อมใช้งาน ทดลองเข้า pypi.org ได้
- [ ] ทดสอบกล้อง/ไมค์/โปรแกรมประชุมออนไลน์เรียบร้อย
- [ ] ถ้ามีปัญหาข้อไหนในนี้ **แจ้งผู้สอนล่วงหน้าก่อนวันเรียน** ไม่ต้องรอถึงวันจริง

> 📌 ถ้าติ๊กครบทุกข้อ วันเรียนแค่เปิดเครื่องมาแล้วติดตั้งตามคลาสได้เลย ไม่ต้องเตรียมอะไรเพิ่ม
